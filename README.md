# SPDIM

SPDIM, or Shattered Pixel Dungeon in Minecraft, is a Minecraft Mod based on an open-source traditional roguelike dungeon game called Shattered Pixel Dungeon, developed by [Evan Debenham](https://github.com/00-Evan) under GNU GPLv3 license. This Mod aims to mimic or redesign items in Shattered Pixel Dungeon to Minecraft world.

Source code of Shattered Pixel Dungeon: [Shattered Pixel Dungeon](https://github.com/00-Evan/shattered-pixel-dungeon)

# List of items implemented from Shattered Pixel Dungeon:
* Wands
  * Wand of Blast Wave
  * Wand of Lightning
  * Wand of Fireblast
  * Wand of Regrowth
* Artifacts:
  * Timekeeper's Hourglass
  * Chalice of Blood

Description of all items can be found in document.txt.

Official releases are in the release section of the repository.
